# HRM

## Technologies and frameworks

- Java 8
- Spring boot 2.4.0
- JWT
### Hướng dẫn cài đặt
1. Git clone URL repo

3. Chỉnh sửa lại datasource của database ở application.properties
4. Run ứng dụng
5. Test API bằng account: username: manager
   password: 123456
